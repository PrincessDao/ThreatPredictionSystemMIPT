# Аналитическая система прогнозирования угроз

## Описание

Демонстрация веб интерфейса для аналитического приложения

## Установка и запуск

1. **Развертывание протита веб-интерфейса**

bash```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
stream run app.py
```

2. **Развертывание аналитического приложения**

bash```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 main.py
```
